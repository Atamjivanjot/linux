# Process Management (Day 4)

## Commands Learned

- ps → view processes
- top → real-time processes
- kill → stop process
- htop → advanced monitoring (if installed)

## Practice

ps aux
top
kill <process_id>

## My Understanding

Processes are running programs in Linux.

## SRE Insight

Used for:
- debugging high CPU usage
- killing stuck services
- monitoring system health
