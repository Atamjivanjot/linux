# Linux Basics

## What is Linux?
Linux is an operating system used in servers and cloud environments.

## Commands Learned
- pwd → shows current directory
- ls → lists files
- cd → change directory
- ls -l → to get all the info regarding the permissions as well
- ls -la → to get hidden files as well
